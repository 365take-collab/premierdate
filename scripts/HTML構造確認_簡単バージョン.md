# HTML構造確認 - 超簡単バージョン

## 🎯 やること

ページ上のホテル名をクリックして、そのHTMLコードを確認するだけです！

## 📝 3ステップで完了

### ステップ1: 要素を選択ツールをクリック

開発者ツールが開いたら、左上にある「要素を選択」アイコン（虫眼鏡🔍のようなアイコン）をクリックしてください。

![要素を選択アイコン]

### ステップ2: ホテル名をクリック

ページ上で、**ホテル名**（例：「○○ホテル」）をクリックしてください。

すると、Elementsタブでそのホテル名の部分のHTMLコードが選択されます。

### ステップ3: HTMLコードを確認

選択されたHTMLコードを見てください。以下のような感じになっているはずです：

```html
<h2 class="hotel-name">ホテル名</h2>
```

または

```html
<div class="hotel-item">
  <h2>ホテル名</h2>
  <p>住所</p>
</div>
```

**この `class="..."` の部分（クラス名）をメモしてください！**

## 📸 こんな感じ

1. 開発者ツールを開く
2. 左上の虫眼鏡アイコンをクリック
3. ページ上のホテル名をクリック
4. HTMLコードで `class="..."` を確認
5. クラス名をメモ

## 💬 見つけた情報を教えてください

以下の情報を教えてください：

1. **ホテル名のクラス名**: `class="???"` の `???` の部分
2. **住所のクラス名**: 住所をクリックした時の `class="???"` の部分
3. **ホテル全体を囲む要素のクラス名**: ホテル名の周りにある `<div class="???">` の部分

この情報を教えていただければ、スクリプトを修正します！

## 🆘 困ったら

- HTMLコードがよくわからなくても大丈夫です
- クラス名だけメモしてください
- スクリーンショットを撮って送っていただいてもOKです
