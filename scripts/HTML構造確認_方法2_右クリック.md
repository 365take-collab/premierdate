# HTML構造確認 - 右クリックで簡単に！

虫眼鏡アイコンが見つからない場合の方法です。

## 🎯 超簡単な方法（右クリック）

### ステップ1: ページ上のホテル名で右クリック

https://happyhotel.jp/ のページ上で、**ホテル名の部分を右クリック**してください。

### ステップ2: 「検証」を選択

右クリックメニューが出てきます。その中から **「検証」**（または「Inspect」）をクリックしてください。

### ステップ3: HTMLコードを確認

自動的に開発者ツールが開いて、そのホテル名のHTMLコードが選択されます。

以下のような感じで表示されます：

```html
<h2 class="hotel-name">ホテル名</h2>
```

または

```html
<div class="hotel-item">
  <h2>ホテル名</h2>
</div>
```

**この `class="..."` の部分（クラス名）をメモしてください！**

## 📝 確認する手順

1. **ホテル名で右クリック** → 「検証」を選択
   - `class="???"` の部分をメモ

2. **住所で右クリック** → 「検証」を選択
   - `class="???"` の部分をメモ

3. **ホテル全体の周りで右クリック** → 「検証」を選択
   - `<div class="???">` のような部分をメモ

## 💡 ヒント

- 右クリックした要素が選択されます
- HTMLコードがハイライト（色が変わる）されます
- その要素のクラス名（`class="..."`）を確認してください

## 🆘 まだ分からない場合

HTMLコードがよくわからなくても大丈夫です。以下を教えてください：

1. ホテル名を右クリック→検証 した時のHTMLコードの一部（スクリーンショットでもOK）
2. 住所を右クリック→検証 した時のHTMLコードの一部

この情報を教えていただければ、スクリプトを修正します！
