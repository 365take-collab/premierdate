# Places API有効化の簡単ガイド（素人向け）

## 📋 現在の状況

✅ Geocoding API: 有効化済み（1つは有効になっているとのこと）
⚠️  Places API (New): 有効化が必要

## 🎯 やるべきこと

**Google Cloud Consoleで「Places API (New)」を有効化するだけです！**

## 📝 ステップバイステップ

### 1. Google Cloud Consoleにアクセス

1. ブラウザで以下を開く:
   ```
   https://console.cloud.google.com/
   ```

2. **Googleアカウントでログイン**
   - APIキーを作成したときと同じアカウントでログインしてください

### 2. プロジェクトを選択

1. 画面上部の「プロジェクトを選択」をクリック
2. APIキーを作成したときに使用したプロジェクトを選択

### 3. Places API (New)を有効化

1. 左側のメニューから **「APIとサービス」** をクリック
   - メニューが見つからない場合は、左上の「☰」（ハンバーガーメニュー）をクリック

2. **「ライブラリ」** をクリック
   - 「APIとサービス」の中に「ライブラリ」という項目があります

3. 検索ボックスに **「Places API (New)」** と入力
   - **重要**: 「Places API」ではなく「**Places API (New)**」を選択してください

4. 「Places API (New)」をクリック

5. **「有効にする」** ボタンをクリック
   - 青色のボタンです

### 4. 確認

有効化されると、画面上部に「Places API (New) が有効になりました」というメッセージが表示されます。

### 5. スクリプトを再実行

ターミナルで以下を実行:

```bash
cd premier-date
npm run collect:shisha-bars
```

## ✅ 確認方法

### 有効化されているAPIを確認する方法

1. 「APIとサービス」→「有効なAPI」をクリック
2. 以下の2つが表示されていればOK:
   - ✅ Geocoding API
   - ✅ Places API (New)

## 🆘 困ったときは

### 「Places API (New)」が見つからない場合

- 「Places API」と検索してみてください
- もし「Places API」しかない場合は、それでも試してみてください
- ただし、「Places API (New)」の方が推奨されています

### 「有効にする」ボタンがグレーアウトしている場合

- 既に有効化されている可能性があります
- 「有効なAPI」で確認してください

### 課金アカウントのリンクが求められる場合

- 初回のみ、課金アカウントのリンクが必要です
- 無料枠（月間 $200）があるので、小規模な使用であれば無料です
- クレジットカードの登録が必要ですが、無料枠内であれば課金されません

## 📞 次のステップ

Places API (New)を有効化したら、以下を実行してください:

```bash
npm run collect:shisha-bars
```

これで、ラブホテルから100m以内のシーシャバーを検索してデータベースに追加できます！
