# 東京カレンダーグルメ店収集スクリプト

ラブホテルからの距離条件を満たす東京カレンダー（東カレ）に掲載されているグルメ店を収集するスクリプトです。

## 対象

- 東京カレンダー（https://tokyocalendar.jp/）に掲載されているグルメ店
- ハッピーホテルサイトに掲載されているラブホテルから一定距離以内にある店舗

## 収集する情報

各グルメ店について：

- 店舗名
- 住所
- 緯度・経度
- エリア名
- 価格帯
- 雰囲気
- 客層
- 横並び席の有無
- ホテルまでの距離（徒歩・電車）
- 用途カテゴリ（初デート、誕生日、記念日、カジュアルデート、夜のデートなど）
- 説明文
- 東京カレンダーのURL（参考情報）

## 実装予定

1. 東京カレンダーサイトからグルメ店情報をスクレイピング
2. 各グルメ店の住所から緯度・経度を取得（Geocoding API）
3. 各ラブホテルから一定距離以内のグルメ店をフィルタリング
4. データベースに追加

## 注意事項

- 東京カレンダーサイトの利用規約を確認
- スクレイピングのレート制限に注意
- データの正確性を確認
