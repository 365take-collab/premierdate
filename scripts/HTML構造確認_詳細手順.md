# HTML構造確認 - 初心者向け詳細手順

## 📖 この手順書について

この手順書は、プログラミング初心者の方向けに、開発者ツールを使ってHTML構造を確認する方法を詳しく説明します。

## 🎯 目標

ハッピーホテルサイト（https://happyhotel.jp/）のHTML構造を確認して、以下の情報を見つけること：

1. ホテル一覧の要素（どこに表示されているか）
2. 各ホテルの名前が表示されている要素
3. 各ホテルの住所が表示されている要素
4. 各ホテルの詳細ページへのリンク

## 📝 ステップバイステップ手順

### Step 1: サイトにアクセス

1. ブラウザで https://happyhotel.jp/ にアクセス
2. ページが完全に読み込まれるまで待つ

### Step 2: 開発者ツールを開く

**Windows/Linux の場合:**
- キーボードの `F12` キーを押す
- または `Ctrl + Shift + I` を同時に押す

**Mac の場合:**
- `Cmd + Option + I` を同時に押す
- または右クリックして「検証」を選択

開発者ツールが画面の下または横に表示されます。

### Step 3: Elementsタブを選択

開発者ツールの上部にタブが並んでいます。以下のようなタブがあります：
- Elements（または要素）
- Console（コンソール）
- Network（ネットワーク）
- など

**「Elements」タブ（または「要素」タブ）をクリックしてください。**

### Step 4: HTML構造を確認

Elementsタブを開くと、たくさんのHTMLコードが表示されます。

#### 4-1. HTMLコードを確認する

HTMLコードは、`<div>`、`<a>`、`<h2>` などのタグで構成されています。

例えば：
```html
<div class="hotel-list">
  <div class="hotel-item">
    <h2>ホテル名</h2>
    <p>住所</p>
  </div>
</div>
```

#### 4-2. ページ上の要素を選択する

**方法A: 要素選択ツールを使う（おすすめ）**

1. 開発者ツールの左上に「要素を選択」アイコン（虫眼鏡のようなアイコン）があります
2. そのアイコンをクリック
3. ページ上のホテル名や住所の部分にマウスを乗せると、その部分がハイライトされます
4. クリックすると、Elementsタブでその要素が選択されます

**方法B: 右クリックで検証**

1. ページ上のホテル名や住所の部分で右クリック
2. メニューから「検証」（または「Inspect」）を選択
3. Elementsタブでその要素が選択されます

### Step 5: クラス名やIDを確認

要素が選択されると、Elementsタブで以下のような情報が表示されます：

```html
<div class="hotel-item" id="hotel-123">
  <h2 class="hotel-name">ホテル名</h2>
  <p class="hotel-address">東京都渋谷区...</p>
  <a href="/hotel/detail/123">詳細を見る</a>
</div>
```

**確認すべき情報：**

1. **クラス名（class）**: `class="hotel-item"` のような部分
2. **ID**: `id="hotel-123"` のような部分（あれば）
3. **タグ名**: `<div>`、`<h2>`、`<p>`、`<a>` など

### Step 6: 情報をメモする

見つけた情報をメモします。以下のような表にまとめると便利です：

| 項目 | 値 | 例 |
|------|-----|-----|
| ホテル一覧のコンテナ | クラス名 | `.hotel-list` |
| 各ホテルアイテム | クラス名 | `.hotel-item` |
| ホテル名 | タグ + クラス名 | `h2.hotel-name` |
| 住所 | タグ + クラス名 | `p.hotel-address` |
| リンク | タグ | `a` |

### Step 7: 複数のホテルを確認する

1つのホテルだけではなく、複数のホテルを確認して、共通の構造を見つけます。

- 全てのホテルが同じクラス名を使っているか？
- HTML構造が同じか？

## 💡 よくある質問

### Q1: HTMLコードが多すぎてわからない

**A:** まず「要素を選択」ツール（虫眼鏡アイコン）を使って、ページ上の実際の要素を選択すると、その部分のHTMLだけがハイライトされます。

### Q2: クラス名が複数ある

**A:** クラス名は複数あることがあります。例：`class="hotel-item active"`。この場合、最初のクラス名（`hotel-item`）を使うことが多いです。

### Q3: どのクラス名を使えばいいの？

**A:** 
- ホテル一覧全体を囲んでいる要素のクラス名
- 各ホテルを囲んでいる要素のクラス名
- ホテル名が表示されている要素のクラス名
- 住所が表示されている要素のクラス名

これらを確認してください。

### Q4: クラス名が英語じゃない

**A:** 日本語のクラス名もあるかもしれませんが、通常は英語です。例：`hotel-list`、`item`、`name` など。

## 📸 スクリーンショットの取り方（参考）

見つけた情報をスクリーンショットで保存しておくと、後で確認しやすいです：

- **Windows**: `Windows + Shift + S`
- **Mac**: `Cmd + Shift + 4`

4. 住所の要素（タグ + クラス名）

この情報を教えていただければ、スクリプトを修正します！
