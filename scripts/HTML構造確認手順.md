# HTML構造確認手順

ハッピーホテルサイト（https://happyhotel.jp/）のHTML構造を確認する手順です。

## 方法1: デバッグスクリプトを使用（推奨）

### 1. デバッグスクリプトを実行

```bash
npm run debug:html
```

### 2. 確認する内容

スクリプトは以下を行います：
- ブラウザを自動で開く（headless: false）
- ページのHTML構造を表示
- 一般的なセレクターをテストして、どの要素が存在するか確認
- スクリーンショットを保存
- ホテル関連のリンクを表示

### 3. ブラウザで確認

ブラウザが開いたら：
1. 開発者ツールを開く（F12 または右クリック > 検証）
2. Elements タブでHTML構造を確認
3. ホテル一覧の要素を選択して、クラス名やIDを確認

### 4. コンソール出力を確認

ターミナルに出力される情報：
- ページのHTML（最初の5000文字）
- テストしたセレクターの結果
- ホテル関連のリンク

## 方法2: 手動でブラウザで確認

### 1. ブラウザでサイトにアクセス

https://happyhotel.jp/ にアクセス

### 2. 開発者ツールを開く

- **Windows/Linux**: `F12` または `Ctrl + Shift + I`
- **Mac**: `Cmd + Option + I`

### 3. Elements タブでHTML構造を確認

1. 開発者ツールの **Elements** タブを開く
2. ページ上のホテル一覧の要素をクリック（または右クリック > 検証）
3. HTML構造を確認し、以下の情報をメモ：
   - ホテル一覧のコンテナ要素のクラス名（例: `.hotel-list`）
   - 各ホテルアイテムのクラス名（例: `.hotel-item`）
   - ホテル名を表示する要素のクラス名
   - 住所を表示する要素のクラス名
   - リンクの構造

### 4. Console タブで確認

1. **Console** タブを開く
2. 以下のコマンドを実行して要素を確認：

```javascript
// ホテル一覧の要素を探す
document.querySelectorAll('[class*="hotel"]')

// すべてのクラス名を確認
Array.from(document.querySelectorAll('*')).map(el => el.className).filter(Boolean)

// リンクを確認
document.querySelectorAll('a[href*="hotel"]')
```
