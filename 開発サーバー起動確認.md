# 開発サーバー起動確認

## 開発サーバーの起動

以下のコマンドで開発サーバーを起動します：

```bash
cd date-guide
npm run dev
```

開発サーバーは `http://localhost:3000` で起動します。

## 確認すべきポイント

### 1. サーバーの起動確認

- ✅ コンソールに「Ready」と表示される
- ✅ `http://localhost:3000` にアクセスできる

### 2. 基本的なページの確認

- **トップページ**: `http://localhost:3000`
- **ログインページ**: `http://localhost:3000/login`
- **検索ページ**: `http://localhost:3000/search`
- **ランキングページ**: `http://localhost:3000/ranking`

### 3. データベース接続の確認

- 認証機能（ログイン・サインアップ）が動作するか
- 店舗データが取得できるか
- APIエンドポイントが正常に動作するか

### 4. エラーの確認

- コンソールにエラーが表示されていないか
- ブラウザのコンソールにエラーが表示されていないか
- データベース接続エラーが発生していないか

## トラブルシューティング

### ポート3000が使用中の場合

```bash
# 別のポートで起動
PORT=3001 npm run dev
```

### 環境変数が読み込まれない場合

`.env`ファイルが正しく配置されているか確認：

```bash
cat .env
```

### Prisma Clientのエラーの場合

```bash
npm run db:generate
```

## 次のステップ

開発サーバーが正常に起動したら：

1. **認証機能のテスト**: ユーザー登録・ログイン機能の動作確認
2. **店舗データの投入**: 初期データの作成
3. **APIエンドポイントのテスト**: 店舗検索、レビュー投稿などの動作確認
4. **UI/UXの確認**: デザイン、レイアウト、レスポンシブ対応の確認
