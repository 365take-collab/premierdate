# ドメイン設定 - クイックスタートガイド

## 🚀 5分で完了！ドメイン設定の手順

### ステップ1: Vercelでドメインを追加（2分）

1. https://vercel.com/dashboard にアクセス
2. `premier-date` プロジェクトを選択
3. **Settings** → **Domains** → **Add Domain**
4. `premierdate.jp` を入力して **Add**

**ネームサーバー情報をコピー**:
- `ns1.vercel-dns.com`
- `ns2.vercel-dns.com`

---

### ステップ2: ドメイン取得サービスでネームサーバーを設定（2分）

1. ドメイン取得サービスの管理画面にログイン
2. **ドメイン一覧** → **premierdate.jp** を選択
3. **ネームサーバー設定** → **カスタムネームサーバー**
4. 上記のネームサーバーを入力して保存

---

### ステップ3: Vercel環境変数を設定（1分）

**Settings** → **Environment Variables** で以下を追加：

| Key | Value | Environment |
|-----|-------|-------------|
| `NEXTAUTH_URL` | `https://premierdate.jp` | Production ✅ |
| `NEXT_PUBLIC_BASE_URL` | `https://premierdate.jp` | Production ✅ |
| `SMTP_FROM` | `noreply@premierdate.jp` | Production ✅ |

---

### ステップ4: デプロイを再実行

**Deployments** → 最新のデプロイメント → **Redeploy**

または、GitHubにプッシュ：

```bash
git add .
git commit -m "ドメイン設定を追加"
git push
```

---

### ステップ5: 反映確認（24-48時間後）

```bash
./check_domain.sh
```

または、ブラウザで `https://premierdate.jp` にアクセス
