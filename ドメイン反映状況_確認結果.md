## ✅ 確認・対処方法

### 1. お名前.comでネームサーバー設定を再確認

1. **お名前.comにログイン**: https://www.onamae.com/
2. **ドメイン一覧** → **premierdate.jp** を選択
3. **ドメイン設定** → **ネームサーバー設定** を開く
4. 以下のネームサーバーが設定されているか確認：
   - **ネームサーバー1**: `ns1.vercel-dns.com`
   - **ネームサーバー2**: `ns2.vercel-dns.com`
5. 設定が正しく保存されているか確認

### 2. 設定が正しく保存されていない場合

1. ネームサーバー設定画面を開く
2. **カスタムネームサーバー** または **外部ネームサーバー** を選択
3. 以下のネームサーバーを入力：
   - **ネームサーバー1**: `ns1.vercel-dns.com`（末尾にピリオド不要）
   - **ネームサーバー2**: `ns2.vercel-dns.com`（末尾にピリオド不要）
4. **保存** ボタンをクリック
5. 設定完了のメッセージを確認

### 3. 設定の反映を待つ

- **反映時間**: 通常24-48時間以内に反映されます
- **確認方法**: 以下のコマンドで確認
  ```bash
  dig NS premierdate.jp
  ```
  結果に `ns1.vercel-dns.com` と `ns2.vercel-dns.com` が表示されるか確認
