# お名前.com - ドメイン設定クイックスタート

## 🚀 5分で完了！お名前.comでのドメイン設定

### ステップ1: Vercelでドメインを追加（2分）

1. https://vercel.com/dashboard にアクセス
2. `premier-date` プロジェクトを選択
3. **Settings** → **Domains** → **Add Domain**
4. `premierdate.jp` を入力して **Add**

**ネームサーバー情報をコピー**:
- `ns1.vercel-dns.com`
- `ns2.vercel-dns.com`

---

### ステップ2: お名前.comでネームサーバーを設定（2分）

**方法A: ネームサーバーを変更する方法（推奨）**

1. **お名前.comにログイン**: https://www.onamae.com/
2. **ドメイン一覧** → **premierdate.jp** を選択
3. **ドメイン設定** → **ネームサーバー設定** → **カスタムネームサーバー**
4. 以下のネームサーバーを入力：
   - **ネームサーバー1**: `ns1.vercel-dns.com`（末尾にピリオド不要）
   - **ネームサーバー2**: `ns2.vercel-dns.com`（末尾にピリオド不要）
5. **保存** ボタンをクリック

**方法B: DNSレコードを直接設定する方法（上級者向け）**

詳細は `お名前.com_ドメイン設定手順_premierdate.jp.md` の「方法B」を参照してください。

---

### ステップ3: Vercel環境変数を設定（1分）

**Settings** → **Environment Variables** で以下を追加：

| Key | Value | Environment |
|-----|-------|-------------|
| `NEXTAUTH_URL` | `https://premierdate.jp` | Production ✅ |
| `NEXT_PUBLIC_BASE_URL` | `https://premierdate.jp` | Production ✅ |
| `SMTP_FROM` | `noreply@premierdate.jp` | Production ✅ |
